## 🤖 Getting GitHub Copilot in VS Code

Follow these steps to get GitHub Copilot (AI coding assistant) set up in VS Code:

### 1. Install VS Code
Download and install [Visual Studio Code](https://code.visualstudio.com/) if you haven't already.

### 2. Sign in to GitHub
1. Open VS Code.
2. Click the **Accounts** icon (bottom-left corner) → **Sign in with GitHub**.
3. Authorise VS Code in your browser.

### 3. Install the GitHub Copilot extensions
Either:
- Open this repo in VS Code — it will prompt you to install the **recommended extensions** automatically (accept the prompt).

Or install them manually:
1. Press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (Mac) to open the Extensions panel.
2. Search for **GitHub Copilot** and click **Install**.
3. Search for **GitHub Copilot Chat** and click **Install**.

### 4. Make sure you have a Copilot subscription
- Go to [github.com/features/copilot](https://github.com/features/copilot) and ensure your account has an active Copilot plan (free tier available for individuals).

### 5. Start using Copilot
- **Inline suggestions** appear automatically as you type — press `Tab` to accept.
- **Copilot Chat** — press `Ctrl+Alt+I` (or click the chat icon in the sidebar) to open a chat window and ask questions.

> **Tip:** If Copilot stops working after a VS Code reinstall, just repeat steps 2–3 above — your subscription carries over automatically.
