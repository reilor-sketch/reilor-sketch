## 🤖 Getting GitHub Copilot in VS Code

1. **Install VS Code** – Download from [code.visualstudio.com](https://code.visualstudio.com) if you don't have it.
2. **Sign in to GitHub** – Open VS Code, click the Accounts icon (bottom-left), and sign in with your GitHub account.
3. **Install the GitHub Copilot extension** – Go to the Extensions panel (`Ctrl+Shift+X`), search for **GitHub Copilot**, and click **Install**. Also install **GitHub Copilot Chat** for the chat interface.
4. **Activate your subscription** – Make sure your GitHub account has an active Copilot subscription (free tier available at [github.com/features/copilot](https://github.com/features/copilot)).
5. **Open this project** – When you open this folder in VS Code it will automatically suggest the recommended extensions listed in `.vscode/extensions.json`.

> **Tip:** If Copilot stops responding, try signing out and back in via the Accounts menu, or run **Developer: Reload Window** from the Command Palette (`Ctrl+Shift+P`).
